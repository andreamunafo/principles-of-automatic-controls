TCLab Lab 1: Coding a relay controller

The purpose of this first laboratory session is to verify that you can interface and interact with the TCLab hardware, and familiarize you with the TCLab library. The first exercise will be to code a rudimentary relay (also called ‚Äòon-off‚Äô or thermostat) controller for one of the two heaters.

Before you begin, you should be familiar with the following reading meterials:

  • On-Off Control in Section 1.6 of Feedback Control for Scientists and Engineers
