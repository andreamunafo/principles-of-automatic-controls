def routh_array(coefficients):
    n = len(coefficients)
    routh = []

    # First two rows
    r1 = [coefficients[i] for i in range(0, n, 2)]
    r2 = [coefficients[i] for i in range(1, n, 2)]
    
    routh.append(r1)
    routh.append(r2 + [0] * (len(r1) - len(r2)))  # Padding zeros

    # Other rows
    for i in range(2, n):
        row = []
        for j in range(len(r1) - 1):

            first = routh[i-1][0]
            upper = routh[i-2][j+1]
            left = routh[i-1][j+1] if j+1 < len(routh[i-1]) else 0

        routh.append(row + [0] * (len(r1) - len(row)))  # Padding zeros

        # Check for row of zeros
        if all(r == 0 for r in row):
            print("Row of zeros detected. Special procedure needed.")
            return routh

    return routh

dove stiamo introducendo un elemento di ritardo $_D $.

L‚Äôequazione caratteristica √®:

\[ 1+G(s)H(s) = 1 + \frac{Ke^{-s\tau_D}}{s(s+1)} = 0 \]

  • La presenza di ritardo rende l‚Äôequazione caratteristica non polinomiale.

  • Tuttavia, approssimando il termine di ritardo $e^{-s_D} $ utilizzando l‚Äôapprossimazione del primo ordine di Pad√©, possiamo tornare alla forma polinomiale, consentendo l‚Äôapplicazione del criterio di Routh.

  • Pi√π specificamente, un‚Äôapprossimazione approssimativa di \(e^{-s\tau_D}\) √®:

\[ e^{-s\tau_D} \approx 1 - s\tau_D \]

Ora abbiamo due parametri sconosciuti: \(K\) e \(T\).

Esercizio: applica il criterio di stabilit√† di Routh all‚Äôequazione caratteristica approssimata che coinvolge $K $ e \(T\). Determinare l‚Äôintervallo di stabilit√† per questi parametri.

\[ Ts^3 + (1+T)s^2 + (1-KT)s + K = 0 \]

\[\begin{array}{c|cc} s^3 & T & 1-KT \\ s^2 & 1+T & K \\ s^1 & \frac{1+T-2KT-KT^2}{1+T} & 0 \\ s^0 & K & \\ \end{array}\]

  • \(T>0\) (fisicamente ha senso) -\(K>0\)
  • \(\frac{1+T-2KT-KT^2}{1+T} >0\) \(\Rightarrow\) \(1+T-KT(T+2)>0\) \(\Rightarrow\) \(KT = \frac{ T+2}{T+1}\) (caso marginale).
